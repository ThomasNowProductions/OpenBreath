# Keyboard Shortcuts

This document provides a comprehensive guide to all keyboard shortcuts available in the BreathSpace application.

## Main Exercise List Screen

### Navigation
- **↑ / ↓** - Navigate up and down through the exercise list
- **Enter / Space** - Select the currently highlighted exercise
- **/** - Focus the search bar
- **Escape** - Exit search focus and return to exercise list
- **Ctrl/Cmd + S** - Open Settings screen
- **?** - Show keyboard shortcuts help

### Focus Management
- Use arrow keys to move between exercises
- Selected exercises are highlighted with a colored border and background
- When search is focused, Escape returns focus to the exercise list

## Exercise Detail Screen

### Navigation & Selection
- **↑ / ↓** - Navigate between start button and version buttons
- **Enter / Space** - Activate the currently selected button
- **Escape** - Go back to the exercise list

### Exercise Versions
- **1** - Select Short version
- **2** - Select Normal version  
- **3** - Select Long version

### Focus Management
- Version buttons are highlighted when selected via keyboard
- Start button is highlighted when selected (index 0)

## Settings Screen

### Navigation
- **↑ / ↓** - Navigate through settings options
- **Enter / Space** - Select the currently highlighted setting
- **Escape** - Go back to the previous screen

### Settings Control
- Navigate between theme, language, view mode, and other settings
- Each option can be activated with Enter or Space

## Exercise Execution Screen

### Exercise Control
- **P** - Toggle play/pause during exercise
- **R** - Restart the current exercise
- **Escape** - Stop exercise and return to detail screen

### Quick Access
- **S** - Open Settings without stopping the exercise
- **M** - Cycle through music modes (Off → Nature → LoFi → Piano)

### Context Awareness
- All keyboard shortcuts work during exercise execution
- Settings opened from exercise screen have "Stop Exercise" option
- Music changes apply immediately without restarting

## Visual Feedback

### Focus Indicators
- **Blue border and background** - Indicates keyboard focus on buttons
- **Highlighted items** - Shows current selection in lists
- **Smooth scrolling** - Auto-scrolls to keep selected items visible

### Progressive Disclosure
- **Help dialog** - Available via "?" key with complete shortcut reference
- **Visual consistency** - Same focus styling across all screens
- **Responsive design** - Keyboard navigation adapts to different screen sizes

## Global Behavior

### Accessibility
- Full keyboard navigation support throughout the app
- Logical tab order and intuitive arrow key navigation
- Consistent keyboard shortcuts across similar functions

### Cross-Platform
- **Ctrl** for Windows/Linux, **Cmd** for macOS where applicable
- Standard keyboard conventions followed
- Fallback touch/gesture interactions always available

### Performance
- Smooth animations and transitions when navigating with keyboard
- Immediate visual feedback for all keyboard actions
- No lag or delay in keyboard responsiveness

## Tips & Tricks

1. **Quick Exercise Start**: Use arrows + Enter to start exercises without mouse
2. **Version Selection**: Press 1/2/3 keys on detail screen for instant version selection
3. **Search & Navigate**: Press "/" to search, Escape to return to list
4. **Exercise Control**: Use P to pause, R to restart during exercises
5. **Settings Access**: Ctrl/Cmd+S from anywhere to access settings

## Troubleshooting

### If keyboard doesn't work:
1. Ensure the app window has focus
2. Try clicking anywhere in the app first
3. Check that no other dialogs are blocking input
4. Restart the app if issues persist

### Focus Issues:
- Press Escape multiple times to exit nested dialogs
- Use mouse click to reset focus if needed
- Keyboard focus is automatically managed by the app
